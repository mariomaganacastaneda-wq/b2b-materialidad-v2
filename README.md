# B2B_Materialidad

## Propósito
Este proyecto está diseñado para asegurar la **Materialidad** y la **Fecha Cierta** de las operaciones B2B bajo el marco fiscal mexicano (Art. 69-B CFF y NOM-151).

## Objetivos
- [ ] Implementar almacenamiento inmutable (WORM) para evidencias.
- [ ] Integrar firma electrónica avanzada (e.firma).
- [ ] Automatizar la validación de listas negras del SAT (69-B).
- [ ] Generar expedientes de defensa listos para auditoría.

## Stack Tecnológico
- **Base de Datos:** Supabase (PostgreSQL)
- **Infraestructura:** MCP para automatización técnica.
- **Entorno:** Local Workspace en `c:\Proyectos\Memoria\B2B_Materialidad`
